# Customer Lifetime Value & Sales Analytics

## Project Overview
This project analyzes real sales transaction data to identify high-value customers, repeat buyers, churn-risk customers, and future customer lifetime value.

## Business Questions Solved
- Who are top customers?
- Which customers may churn?
- Who are repeat high-value buyers?
- How can we segment customers?
- What future revenue can customers generate?

## Methods Used
- Python
- Pandas
- NumPy
- Scikit-learn
- KMeans Clustering
- Linear Regression
- BG/NBD Model
- Gamma-Gamma CLV Model
- Plotly / Matplotlib / Seaborn

## Files
- clv_customer_segmentation.ipynb
- probabilistic_clv_prediction.ipynb

## Key Outputs
- Customer Segments
- Churn Risk Detection
- Predicted CLV
- Repeat Buyer Analysis
